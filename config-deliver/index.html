# configDeliver

配置分发服务端

(opens new window) (opens new window)

# 功能

  • 提供一个简单的 http 服务端，用于配置分发
  • 可同时提供多个分发服务
  • 多个 ID 可对应同一分发目录
  • 支持单文件分发或者目录分发
  • md5 校验,避免重复分发
